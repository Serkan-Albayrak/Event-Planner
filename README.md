# Event Planner

📌 Proje Amacı

Bu proje, modern JavaScript kütüphaneleri kullanarak CRUD işlemlerini gerçekleştiren bir web uygulaması geliştirmek amacıyla hazırlanmıştır.
React, Tailwind ve Vite kullanılarak modern frontend yapısı oluşturulmuştur.

Modern ve basit bir etkinlik planlama uygulaması.  
Kullanıcılar etkinlik ekleyebilir, düzenleyebilir, silebilir ve listeleyebilir.

## 🚀 Canlı Demo
Netlify: https://serkan-eventplanner.netlify.app/

---

## 🛠 Kullanılan Teknolojiler
- React (Vite)
- Tailwind CSS
- JavaScript (ES6)
- LocalStorage (veri kalıcılığı)

---

## ✨ Özellikler
- Etkinlik ekleme
- Etkinlik listeleme
- Etkinlik güncelleme
- Etkinlik silme
- Tüm etkinlikleri temizleme
- Arama özelliği
- LocalStorage ile kalıcı veri
- Modern ve responsive arayüz
